Resources for R

What is R?

According to Wikipedia, R is "a programming language and software environment for statistical computing and graphics. The R language is widely used among statisticians and data miners for developing statistical software and data analysis." To data scientists, it's bread or butter.

How do I install R?

R is available for Windows, OS X, and Linux. Did we mention it's also free? Head over to the R Project to download and install. Some people refer to work in base R (or in the terminal), but others like to work in an IDE. If you're among the latter, you might want to try out R Studio.

How do I learn R?

While the aRt Challenge will help you learn R, we throw you immediately into the deep end of the pool. If you're a beginner, we recommend the following resources.

WEB COURSES

(1) Try R: learn basic R programming via a pirate-themed interactive console (free)
(2) Data Camp: for those serious about data science, this website offers a full set of R courses from beginner to advanced (some free; monthly subscription)

WEB REFERENCES

(1) Quick-R: a comprehensive reference guide to basic and intermediate R programming; with lots of helpful examples -- perfect for beginners
(2) UCLA's R Resources: a wiki containing lots of notes and examples
(3) R Tutorial: in addition to explaining basic R programming, this website introduces statistical concepts

BOOKS

(1) The Art of R Programming (Matloff)
(2) R Cookbook (Teetor)
(3) R in Action (Kabacoff)
